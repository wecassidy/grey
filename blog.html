A Quick Guide to Mathematical Proofs

by Tungsten, Mar 23, 2014, 9:33 pm

Direct Proof
Direct proofs are like cooking. You put stuff in a bowl and apply operations to the stuff in the bowl until it comes out yummy.

How to do it:
  1. Turn the thing you are trying to cook prove equal.
  2. Do stuff to the two sides until you end up with two numbers that are the same.
  3. Write QED at the end.

Proof by Induction
Proving stuff by induction is a lot like building a domino chain. You put the last domino in place, line all of the dominoes up, and then push the first one over.

How to do it:
  • Method 1:
    1. Prove the "base case" (show that the first domino exists).
    2. Make the somewhat dubious assumption that the case at works (assume the second-last domino exists).
    3. Try to turn the case at into the case at (hope that if you push over the last domino, the next one will fall over).
    4. Write QED at the end.
  • Method 2:
    1. Prove the "base case" (show that the first domino exists).
    2. Hope that the cases up to work (hope that the middle of the domino chain exists).
    3. Turn the case at into the case at (push over the last domino and hope that the previous domino exists, and is knocked over).
    4. Write QED at the end.
  • Method 3:
    1. Prove the "base case" (show that the first domino exists).
    2. Note that if there is one domino, there must be a box of dominoes. Thus, the rest of the cases (which are dominoes, remember) are true and you can go build a real domino chain instead of this silly hypothetical one.
    3. Write QED at the end so that your math teacher/professor/tutor/figure of authority doesn't get mad at you for not finishing your proof properly.

Proof by contradiction is, in some respects, similar to making a mistake while solving a problem. The only difference is that you make the mistake on purpose.

  1. Scanner s = new Scanner(new File(fileName));

  4. for (; s.hasNext(); file += s.nextLine() + "\n");
  5.  
  6. String[] docs = file.split("\n\\*\n"); // Split the file by each asterisk (including the newlines before and after it)
  7. for (int i = 0; i < docs.length; i ++) {
  8.     for (String word : docs[i].split(" ")) { // Go through each word in the current document as split by a space
  9.         array[i] = word
  10.     }
  11. }


Code:
  1. f = open(fileName)
  2.  
  3. fStr = f.read()
  4. array = [word for line in fStr.split("\n*\n") for word in line.split()]
  5. f.close()


Or even more compact:
Code:
  1. f = open(fileName)
  2.  
  3. array = [word for line in f.read().split("\n*\n") for word in line.split()]
  4. f.close()


Or in a one-liner:
Code:
  1. array = [word for line in open(fileName).read().split("\n*\n") for word in line.split()]


The only problem with the last one is that it doesn't close the file. But all of the Python versions are way more readable than the Java equivalent.
